# History v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refactor `history.jsonl` schema to v2 — drop `text_stats.chars`, rename `asr.raw` → `asr.text`, add top-level `text` (explicit final dispatched text), add `asr.sessions[].audio_ms`. Bump `version: 1` → `2`. Multi-session writer and VAD are out of scope; `sessions[]` always has length 1 until that work lands.

**Architecture:** Pure schema/refactor work. No new modules. Type changes ripple through `state/history.rs` → `voice/finish.rs` → `tui/*` / `ipc/*`. Live recording stats (`StateSnapshot` / overlay) drop `chars` and keep `words` for symmetry.

**Tech Stack:** Rust, `serde` / `serde_json`, `unicode-segmentation` (already in deps), `time` crate (RFC 3339).

**Reference design:** `docs/superpowers/specs/2026-06-15-multi-session-asr-and-history-v2-design.md`

```rust
pub struct HistoryRecord {
    pub version: u8,

    #[serde(with = "time::serde::rfc3339")]

    pub app: Option<String>,
    pub text: String,

    pub asr: AsrHistory,

    #[serde(skip_serializing_if = "Option::is_none")]

}

impl HistoryRecord {
    /// 最终上屏文本。等同 `record.text`（保留 method 给调用者迁移期间用，未来可删）。
    pub fn final_text(&self) -> &str {
        &self.text

    pub fn text_stats(&self) -> TextStats {
        self.text_stats

}
```

```rust
async fn dispatch_with_post_chain(
    segments: &[SegmentCapture],
    auto_paste: bool,
    app_context: &post::AppContext,
    params: &SessionParams,
) -> Result<
    (
        String,                       // dispatched final text (== "" if nothing to dispatch)
        Vec<PipelineStepHistory>,
        HistoryStatus,
        Option<HistoryError>,
    ),
    HistoryError,
> {
    let segment_texts: Vec<String> = segments.iter().map(|s| s.text.clone()).collect();
    let raw_text: String = segment_texts.concat();
    if raw_text.is_empty() {
        crate::debug_println!("[shuo] (空识别结果，跳过 dispatch)");
        return Ok((String::new(), Vec::new(), HistoryStatus::Canceled, None));
    }
    let initial = PipelineText::new(raw_text, segment_texts);
    overlay_send(
        params,
        OverlayCmd::SetState {
            state: OverlayState::Thinking,
        },
    );
    let (out, steps) = post::run_chain(
        &params.post_chain.processors,
        initial,
        app_context,
        Duration::from_millis(params.post_timeout_ms),
    )
    .await;
    for step in &steps {
        // (unchanged: notice routing)
        match step.status {
            PipelineStepStatus::Error | PipelineStepStatus::Timeout => {
                let text = match step.status {
                    PipelineStepStatus::Timeout => {
                        crate::t!("notice.step_timeout", name = step.name)
                    }
                    _ => crate::t!("notice.step_failed", name = step.name),
                };
                overlay_send(
                    params,
                    OverlayCmd::Notice {
                        text,
                        ttl_ms: NOTICE_TTL_MS,
                    },
                );
            }
            PipelineStepStatus::Ok | PipelineStepStatus::Skipped => {}
        }
    }
    crate::debug_println!("[shuo] ✓ 最终: {}", out.text);
    let dispatched = out.text.clone();
    let pipeline = steps.into_iter().map(PipelineStepHistory::from).collect();
    if let Err(e) = dispatch::dispatch(&out.text, auto_paste) {
        eprintln!("[shuo] ❌ 剪贴板写入失败: {e:#}");
        return Ok((
            dispatched,
            pipeline,
            HistoryStatus::Error,
            Some(HistoryError {
                kind: "dispatch".to_string(),
                msg: format!("{e:#}"),
            }),
        ));
    }
    Ok((dispatched, pipeline, HistoryStatus::Submitted, None))
}
```
